DASHBOARD DE VENTAS CON IA

Construido en un día · 6h/semana recuperadas

La mayoría de dueños de negocio lleva sus ventas en una hoja de cálculo. Este sistema las registra solo — cada vez que terminas una llamada, todo aparece automáticamente en el dashboard.

Las 3 herramientas

Fathom
Transcribe la llamada automáticamente cuando termina
n8n
Procesa la transcripción y la envía a Claude para análisis
Claude + Airtable
Claude extrae los datos y los registra en Airtable para el dashboard

Flujo automático

Llamada termina → Fathom transcribe → n8n procesa → Claude analiza → Dashboard listo
